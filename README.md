The program is designed to evaluate robust prognosability and its stealthy counterpart on randomly generated DES models under observation attacks. For each run, a plant with a randomly selected number of states is first generated so that it remains prognosable in the nominal setting. Then, an attack scenario is randomly assigned by specifying insertion and erasure event sets, which are used to construct an attack-aware verifier for checking robust prognosability against all attacks. To further assess robustness against stealthy attacks, the verifier is reduced to a stealthy verifier by removing behaviors that violate stealthiness, and the corresponding RSDA property is then tested. During the experiment, the program records the plant size, event-set sizes, verifier sizes, and computation times, so that both the logical verification results and the computational complexity can be compared across different randomly generated systems.
